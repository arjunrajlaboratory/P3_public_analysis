README for analysisScripts/

Make sure to have MATLAB_2019a and rajlabimagetools changeset 775fd10 (https://bitbucket.org/arjunrajlaboratory/rajlabimagetools/wiki/Home) installed (or MATLAB_2017a and rajlabimagetools changeset a2c6ac5 (https://github.com/arjunrajlaboratory/rajlabimagetools/).

Make sure to have R_3.6.1 installed, and all packages versions listed below (and in the sessionInfo.rds file in this directory). Make sure all such packages are accessible in your default R .libPaths().

Run all scripts by modifying variables in analyzeAll.sh, and then execute from one level up, with the following usage:
$ pwd
~/P3_public_analysis # can be whatever top-level project directory in which you place analysisScripts/ and all the other necessary folders specified in P3_public_analysis/README.md
$ bash analysisScripts/analyzeAll.sh
